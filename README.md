🎓 Enhanced EduTrack – Attendance Dashboard

Enhanced EduTrack is a contemporary, interactive teacher dashboard that makes attendance tracking, course management and academic monitoring easier. It was developed with HTML, CSS, JavaScript and Chart.js, offering a clean and responsive user interface for handling classroom data efficiently.
