🎓 Enhanced EduTrack – Attendance Dashboard

Enhanced EduTrack is a modern, interactive teacher dashboard designed to simplify attendance tracking, course management, and academic monitoring. Built using HTML, CSS, JavaScript, and Chart.js, it provides a clean and responsive interface for managing classroom data efficiently.

🚀 Features

🔐 Teacher Login Interface

📊 Real-time Attendance Dashboard (Pie & Bar Charts)

🧑‍🎓 Student Attendance Management (Present / Absent / Late)

📚 Course Overview & Detailed Course View

💬 Messages Section

📁 Documents Management

📅 Weekly Schedule View

📱 Fully Responsive Design

🛠️ Technologies Used

HTML5

CSS3

JavaScript (Vanilla JS)

Chart.js (for data visualization)

📂 Project Structure
index.html   # Main application file (UI + Logic)

This project runs entirely on the frontend and does not require a backend server.

▶️ How to Run

Download or clone the repository:

git clone https://github.com/your-username/enhanced-edutrack.git

Open the project folder.

Double-click index.html
or
Open it in your browser using Live Server (VS Code recommended).

🔑 Demo Login

You can enter any Teacher ID and Password to access the dashboard (authentication is simulated for demo purposes).
